This is a  plugin I created for the Serviced City Pads website to create a bookings management system for their clients and operators. 

== Installation ==
Installation: 
1. Upload the plugin folder using an FTP client to your host to the plugins directory (yourdomainname.co.uk/wp-content/plugins)
2. Extract the plugin
